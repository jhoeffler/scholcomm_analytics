Hybrid Open Access Publishing by Three Selected Major Publishers

In this article we want to provide answers to several research questions related to the access in different databases of Hybrid Open Access Publications published by selected editorials, namely: Royal Society of Chemistry (RSC), Cambridge University Press (CUP) and SAGE Publications.

Jan H. Höffler (based on the pre-existing HOAD approach) https://replication.uni-goettingen.de/wiki/index.php/User:Jan_H._Höffler
06-02-2020

Research Questions:

  • Of how many publishers, journals and articles can data about hybrid open access publications be obtained for the years 2013-2020?
  • What percentage of publishing in academic journals occurs in the form of hybrid open access?
  • What are the limitations to determining this information?

Data Sources:

  • Open APC1
  • CrossRef
  • Unpaywall2
  • Dissemin3
  • oaDOI
  • Directory of Open Access Journals (DOAJ)

Methods:

  • Obtaining information about hybrid journals from the two datasets provided by the Open APC (Article Processing Charge) Initiative, a core dataset and a second one on transformative agreements. The datasets can be concatenated as they do not overlap.
  • Search for license data for determined journals in CrossRef
  • Use approach based on dissemin and others to determine open access from license data
  • Remove data about journals after4 they have turned to full open access in observation period using data from the Directory of Open Access Journals
  • Compare information with those provided about open access publications by Unpaywall

library(dplyr)
#Unique number of licenses and journals
#Unique number of journals
#Here distinct takes a data frame and returns one where if a row has the same 3 elements, it only appears once
#Example: 2013, Springer, MIT_license only appears once
journals<-o_apc %>% distinct(publisher, journal_full_title,period,.keep_all = TRUE)
#xtabs creates a matrix which counts the number of cells that have a disctinct pair of column names
#example, it counts all cells that have the pair springer, 2014
unique_journal<-as.data.frame.matrix(xtabs(~ publisher+period,
           data = journals))

#Unique number of licenses
license<-o_apc %>% distinct(publisher, license_ref,period,.keep_all = TRUE)
unique_license<-as.data.frame.matrix(xtabs(~ publisher+period,
           data = license))

lic_jour<-unique_journal
for (row in 1:nrow(unique_journal)){
  for (col in 1:ncol(unique_journal)){
    lic_jour[row,col]<-paste(unique_license[row,col],"/",unique_journal[row,col])
  }
}
names<-data.frame(Publisher=row.names(lic_jour))

Summary

Since the approach is based on data of the Open APC Initiative to which only certain institutions primarily from Europe contribute, coverage is limited. With the available data it is not possible to calculate what percentage of the literature is produced by these institutions, and as not even from all participating countries there is data for every year, we know that this is a restriction. What needs to be determined is even if there is only information on a small percentage of articles published, what percentage of hybrid journals per year does this already cover?
A second limitation that becomes apparent is contribution of data about open access licenses to CrossRef. As could be seen clearly in the case of SAGE Publications, coverage is very incomplete and delayed. The information can be complemented with data from Unpaywall. Already the Open APC data covered a significant part of hybrid journals of at least the Royal Society of Chemistry but the license information collected by Open APC is not normalized. While for Cambridge University Press the number of observations is steadil growing in the process of our data transformation, for Royal Society of Chemistry and SAGE Publications much information is lost in the process of matching the ISSNs obtained from Open APC to available CrossRef licenses for open access. A finer grained approach to see at which step of the data information the numbers of journals by the publishers is reduced in our data will help to determine the influence of delayed access licenses and journal transformations to full open access. As was seen with manual checks, data quality issues also play a role as in some cases ISSNs are not stored correctly and journals change names and publishers.[^6] The number of different licenses used is steadily growing. Methods were already developed to harmonize license data and we can further contribute to improve on these approaches[^7]. While the continued introduction of new licenses will remain a challenge, we could show that even for some major publishers this number so far has stayed manageably low.
